# MQL5-DK-StdLib-Extension
My extensions of the standard MQL5 library.

DKStdLib extends the MQL5 library with the following functions:
- **Analysis**
- **Bot**
- **Common**
- **Drawing**
- **Logger**. Simplest Logger class for MetaTrader 5 [CDKLogger](Logger.md).
- **NewBarDetector**
- **TradingManager**
